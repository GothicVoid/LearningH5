元素 html
元素类型 无
允许具有的父元素 无
局部属性 manifest
内容 head元素和body元素各一
标签用法 开始标签和结束标签，内含其他元素
是否为HTML5新增 否
在HTML5中的变化 manifest属性是HTML5中新增的。HTML4版本中的属性已不再使用
习惯样式 html { display: block; }
html:focus { outline: none: }

html元素也是创建HTML文档必备的元素之一，它表示文档中HTML部分的开始。
使用html元素:
<!DOCTYPE HTML>
<html>
......此处省略内容和元素......
</html>
